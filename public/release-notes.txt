1. Route Inventory
------------------
The React application contains the following public indexable paths, configured to fallback to index.html in standard Apache hosting via .htaccess:
- /
- /products
- /partners
- /team
- /contact
- /terms-of-use
- /privacy-policy
- /relationship-vista-user-guide
- /relationship-vista-installation-guide

A 404 Not Found dynamic route acts as a catch-all for any unregistered paths.

2. Infrastructure Settings & .htaccess configuration
----------------------------------------------------
- The .htaccess file is securely packaged inside the production bundle (`site-root/`).
- It has been prepared to:
  - Provide an SPA (Single Page Application) fallback. Let the URL rewrite rule handle incoming nested routes.
  - Apply an HTTPS forced redirect.
  - Force non-www `ardira.com` canonical hostname.
